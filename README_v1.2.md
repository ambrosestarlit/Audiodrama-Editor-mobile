## 🚀 クイックスタート

### 1. デプロイ
```bash
# ZIPを解凍
unzip Sound-Bear-Mobile-v1.2.zip

# GitHubにプッシュ
git add .
git commit -m "Add Sound Bear Mobile v1.2"
git push
```

### 2. GitHub Pages設定
1. Settings → Pages
2. Source: main branch
3. 公開URLにアクセス!

### 3. スマホで使う
1. +ボタンで音声ファイル追加
2. トラック追加ボタンでトラック作成
3. ファイルをドラッグしてトラックに配置
4. ▶ボタンで再生!

## 🐛 トラブルシューティング

### iPhoneでファイルが選択できない
→ **v1.2で修正済み!** 最新版をお使いください

### 音が出ない
1. デバイスのボリュームを確認
2. トラックがミュートでないか確認
3. ブラウザをリロード

### 動作が重い
1. 使っていない素材を削除
2. トラック数を減らす(10前後推奨)
3. ブラウザを再起動

## 💡 Tips

### iPhoneでの快適な使い方
1. ホーム画面に追加すると便利!(Safariの共有メニューから)
2. 横画面でも使えますが、縦画面が最適
3. ファイルは事前にiPhone内に保存しておく

### おすすめワークフロー
1. PCで大まかに編集
2. スマホで外出先で微調整
3. PCで最終仕上げ
