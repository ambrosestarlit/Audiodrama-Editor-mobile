# 📱 Sound Bear - クイックスタート

## 🚀 5分で始める!

### 1. ファイルのデプロイ
```bash
# ZIPを解凍
unzip Sound-Bear-Mobile.zip

# GitHubリポジトリにプッシュ
git add .
git commit -m "Add Sound Bear Mobile"
git push
```

### 2. GitHub Pagesで公開
1. GitHubリポジトリの **Settings** → **Pages**
2. Source: **main branch**
3. 公開URL: `https://[username].github.io/[repo-name]/`

### 3. スマホでアクセス!
公開URLにスマホでアクセスすれば、すぐに使えます!

## 📱 スマホでの基本操作

### 素材を追加
1. 画面上部の **+ボタン** をタップ
2. 音声ファイルを選択

### トラックを作る
1. 画面下部の **トラック追加** をタップ

### 音を配置
1. 素材をタップ&ホールド
2. トラックにドラッグ
3. 指を離す

### 再生する
1. 中央の **▶ボタン** で再生!

### 保存する
1. **☰メニュー** → **💾保存**
